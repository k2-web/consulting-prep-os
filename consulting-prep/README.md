# Consulting Prep OS (v2.0)

A modern, interactive platform designed to master PwC consultant interviews.

## Features

* **Knowledge Base**: Curated frameworks (Profitability, Market Entry) and guesstimate guides.
* **AI Coach**: Interactive mock interviews with a "Coaching Mode" for real-time hints.
* **Modern UI**: Premium dark-mode aesthetic with glassmorphism design.

## How to Run Locally

1. Clone the repository.
2. Install dependencies:

    ```bash
    pip install -r requirements.txt
    ```

3. Run the app:

    ```bash
    streamlit run app.py
    ```

## How to Deploy (Streamlit Cloud)

1. Push this code to a GitHub repository.
2. Go to [share.streamlit.io](https://share.streamlit.io/).
3. Connect your GitHub account and select this repository.
4. Click **Deploy**!
